# Bug Bounty

## ROKO Bug Bounty Program

### Rewards

| Severity | Reward | Examples |
|----------|--------|----------|
| Critical | $50,000 - $100,000 | Consensus failure, fund theft |
| High | $10,000 - $25,000 | Time manipulation, DoS |
| Medium | $2,500 - $5,000 | Logic errors, gas issues |
| Low | $500 - $1,000 | UI bugs, minor issues |

### In Scope

#### Core Protocol
- Consensus mechanism
- Time synchronization
- Validator software
- Network layer

#### Smart Contracts
- System contracts
- Time library
- Token contracts
- Governance contracts

#### Infrastructure
- RPC endpoints
- Block explorer
- Official wallets

### Out of Scope
- Third-party integrations
- Social engineering
- Physical attacks
- Already reported issues

### Submission Process
1. Discover vulnerability
2. Document with proof-of-concept
3. Submit to security@roko.network
4. Wait for confirmation (24-48h)
5. Collaborate on fix
6. Receive reward after patch

### Responsible Disclosure
- Do not exploit on mainnet
- Keep findings confidential
- Allow 90 days for fixes
- Coordinated disclosure
