# Validators

## ROKO Network Validators

Validators are the backbone of ROKO's temporal consensus, maintaining nanosecond-precision synchronization.

### Key Responsibilities
- **Time Synchronization**: Maintain atomic clock sync
- **Block Production**: Create temporally-ordered blocks
- **Consensus Participation**: Vote on block validity
- **Network Security**: Protect against temporal attacks

### Validator Types

#### Full Validators
- Run atomic clock hardware
- Participate in consensus
- Earn full rewards

#### Light Validators
- GPS/PTP time sources
- Support network redundancy
- Reduced hardware requirements
